## Getting started for development

### Prerequisite
- [Flutter](https://flutter.dev/) (Currently this project use version 3.3)
- Run `flutter doctor`

### Install package dependencies
```
$ flutter pub get
```

### Run unit tests
```
$ flutter test
```

### With coverage report
```
$ flutter test --coverage
```
